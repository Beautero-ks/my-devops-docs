# 🚀 Introduction : Pipeline CI/CD Multi-Plateforme

## Contexte du Projet

Ce projet démontre l'implémentation d'un pipeline **CI/CD professionnel** capable de déployer automatiquement une application statique sur **deux plateformes distinctes** :

- **Vercel** : Déploiement principal avec CDN mondial
- **GitHub Pages** : Solution de fallback et documentation publique
